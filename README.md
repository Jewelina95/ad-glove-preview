# AD Glove Dashboard Preview

Public live preview of the AD wearable glove dashboard (patient + doctor demo views).

- **Source of truth:** [github.com/CZ114/ainone-dashboard](https://github.com/CZ114/ainone-dashboard) → `docs/ad-dashboard/`
- **Live URL:** https://jewelina95.github.io/ad-glove-preview/
- **What it is:** Static HTML demo with mock patient data (3 patient profiles)
- **Real-time ESP32 data** requires running the FastAPI backend in the source repo locally
