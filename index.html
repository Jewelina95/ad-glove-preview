📡 实时面板需要本机运行

实时 ESP32 传感器数据是 React 应用 + FastAPI backend，无法部署到静态 GitHub Pages。需要在你电脑上跑起来：

1. Clone 代码

git clone https://github.com/CZ114/ainone-dashboard.git
cd ainone-dashboard

2. 启动 backend（终端 1）

cd backend
pip install -r requirements.txt
python run.py
# 监听 http://localhost:8000

3. 启动 frontend（终端 2）

cd frontend
npm install
npm run dev
# 打开 http://localhost:5173

4. 插上 ESP32 板，访问 /dashboard

如果你已经在跑了，点这个直接打开：
